### 1. Работа ESPNOW не только на broadcast
- [ ] Раскомментировать `control.activePeer(storage.config().activePeer())` в `main.cpp`
- [ ] Реализовать страницу `PeersPage` с ComboBox или списком кнопок:
  - Отображение MAC-адресов из `favorites`
  - Выбор активного пира → вызов `control.activePeer(mac)`
  - Добавление своего MAC (кнопка "Add Self")
  - Удаление выбранного пира

### 2. Сброс конфигурации с перезагрузкой
- [ ] В `ConfigManager::reset()` после `save()` добавить `ESP.restart()`
- [ ] Убедиться, что кнопка "Reset" в `ConfigPage` вызывает перезагрузку

### 3. Всплывающие сообщения об ошибках на экране
- [ ] В `DisplayManager` добавить очередь сообщений (например, `static const char* lastError`)
- [ ] Показывать сообщение (ESP‑NOW init fail, Display init fail) поверх UI на 1‑2 секунды

### 4. Версионирование конфигурации
- [ ] Добавить в `Config` поле `uint16_t version` и `static constexpr uint16_t CURRENT_VERSION`
- [ ] В `ConfigManager::load()` проверять версию, при несовпадении сбрасывать на дефолт

### 5. Выбор типа дисплея через флаг компиляции
- [ ] Добавить в `platformio.ini` build flag `-DDISPLAY_TYPE=ST7735` или `SSD1306`
- [ ] В `prelude.hpp` условно подключать драйвер и пины (через `#ifdef`)

### 6. Полноценный MAVLink (приём телеметрии)
- [ ] Добавить приём и отображение `ATTITUDE`, `GLOBAL_POSITION_INT`, `SYS_STATUS`
- [ ] Создать страницу телеметрии (высота, крен, тангаж, напряжение)

### 7. Таймер бездействия и light sleep
- [ ] Добавить `IdleSleepManager` (код выше)
- [ ] Вызывать `reset()` при любом действии пользователя (стики, кнопки)
- [ ] В `loop()` вызывать `IdleSleepManager::update()`
- [ ] После пробуждения переинициализировать ESP‑NOW и восстановить активный пир

---

**Приоритет (порядок выполнения):**
1. Работа ESPNOW не только на broadcast (п.1) – иначе пульт бесполезен.
2. Сброс конфига с перезагрузкой (п.2) – чтобы откатиться при ошибках.
3. Всплывающие ошибки (п.3) – для диагностики.
4. Таймер бездействия (п.7) – чтобы не грелся.
5. Версионирование (п.4), выбор дисплея (п.5), MAVLink телеметрия (п.6) – опционально.
